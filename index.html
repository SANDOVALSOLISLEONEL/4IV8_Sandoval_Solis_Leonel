Objetivo del portafolio

Este portafolio tiene como propósito concentrar y mostrar las prácticas desarrolladas a lo largo de la materia de Programación Web, evidenciando el proceso de aprendizaje progresivo desde la estructura básica de una página HTML hasta la construcción de sistemas con arquitectura cliente-servidor, bases de datos y patrones MVC. Cada práctica representa un avance concreto en el dominio de tecnologías web modernas.

Elementos aprendidos en la materia

  • Estructura semántica de HTML5: uso de etiquetas header, nav, section, article, aside y footer
  • Diseño y estilizado con CSS3: modelo de caja, selectores, posicionamiento y guía de estilos personalizada
  • Formularios HTML con validación del lado del cliente mediante JavaScript
  • Fundamentos de JavaScript: variables, funciones, eventos y manipulación del DOM
  • Resolución de problemas algorítmicos con JavaScript: inversión de palabras, producto escalar y caracteres únicos
  • Funciones avanzadas en JavaScript: parámetros, retorno de valores y reutilización de código
  • Desarrollo de un juego interactivo (rompecabezas) con manejo de eventos y lógica de tablero
  • Arquitectura cliente-servidor: creación de una API REST básica con Node.js y Express
  • Patrón MVC con Express, MySQL y manejo de rutas para múltiples entidades (usuarios, productos, compras)
